# To Do List

A task list project built with `React + TypeScript + Vite`.

## Features

- Add a new task
- Search tasks by title
- Mark a task as completed
- Delete a single task
- Delete all tasks
- Display the number of found tasks
- Save tasks in `localStorage`
- Auto-focus on the add task input
- Scroll the list when there are many tasks

## Run the Project

1. Clone the repository:

```bash
git clone <repository-url>
cd todo-react-ts
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the app in the browser using the URL provided by Vite.
